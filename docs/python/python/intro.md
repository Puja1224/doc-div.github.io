# Introduction to Python

Python is one of the most popular and beginner-friendly programming languages in the world.  
It is known for its simplicity, readability, and powerful capabilities in areas like web development, data science, automation, and artificial intelligence.

---

## Why Learn Python?

- Easy to read and write, perfect for beginners.  
- Used by top companies like Google, Netflix, and Instagram.  
- Huge community and countless open-source libraries.  
- Works on all platforms (Windows, Mac, Linux).
